# 🏋️ Fitness Hub AI

Fitness Hub AI is an AI-powered fitness assistant that generates personalized workout plans, diet recommendations, protein intake suggestions, and motivational guidance using IBM Granite Foundation Models through IBM watsonx.ai.

---

## 🚀 Features

- 🏋️ Personalized workout plans
- 🥗 AI-generated diet recommendations
- 💪 Protein intake suggestions
- 🎯 Goal-based fitness planning
- 🤖 Powered by IBM Granite Foundation Model
- ☁️ Deployed using Streamlit Cloud
- 🌐 Accessible through a web browser

---

## 🛠️ Technologies Used

- Python
- Streamlit
- IBM watsonx.ai
- IBM Granite Foundation Model
- Git & GitHub
- IBM Cloud

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/iamTIMEATER/Fitness_Hub_AI.git
```

Move into the project

```bash
cd Fitness_Hub_AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file using `.env.example`.

Run the application

```bash
streamlit run app.py
```
